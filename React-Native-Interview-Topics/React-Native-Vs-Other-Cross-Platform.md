# React Native vs Other Cross Platform.

React Native is a open source cross platfrom framework developed by meta (formarly facebook) which is used to develop native mobile application for android and ios as well as web and desktop application using javascript and react.

**Advantage of React Native**

- Enable Faster development through Cross Platform Code Reusability
- Reduce development and maintenance time
- Large Community
- Renders Real native UI Components
- Easy Learning Curve for Frontend Web Developer
- Hot Reloading
- Delivers Near Native Performance (Better with new Architecture)
- Cost Effectiveness

**Disadvantage of React Native**

- Relies heavily on third-party libraries
- Requires platform-specific code for complex features
- Can face performance issues in highly complex or graphics-heavy apps

**Other cross platform Frameworks are**

- Flutter
- Lynx
- KMM (Kotlin Multiplatform)
