# React native

React Native is a open source cross platfrom framework developed by meta (formarly facebook) which is used to develop native mobile application for android and ios as well as web and desktop application using javascript and react.

# The code idea was-

- Write once and run anywhere.
- Save time and save money.
- Still get real native performance.

Earlier to devlop mobile application for android and ios we need to use two different framework and two different language to develop mobile application. java and kotlin for android and swift and objective c for ios.

Twice the work, twice the debugging, twice the time consuming
